# Radha Oil Industry Website

🌾 **प्राकृतिक तेल और मसालों का विश्वसनीय स्रोत**

## About / परिचय

Radha Oil Industry is a modern e-commerce website for selling natural oils and spices. The website features a beautiful design with animations, an admin panel for managing products, and a complete shopping cart system.

राधा ऑयल इंडस्ट्री प्राकृतिक तेल और मसालों की बिक्री के लिए एक आधुनिक ई-कॉमर्स वेबसाइट है। इसमें सुंदर डिज़ाइन, एनिमेशन, प्रोडक्ट मैनेज करने के लिए एडमिन पैनल और पूर्ण शॉपिंग कार्ट सिस्टम है।

## Features / विशेषताएं

- ✨ Modern animated homepage with farmer-based logo
- 🛒 Complete shopping cart with Add to Cart functionality
- 📱 Mobile responsive design
- 🔧 Admin panel for managing products, orders, and customers
- 🔄 Real-time sync between admin panel and website
- 📦 7 products: Mustard Oil, Linseed Oil, Wheat Flour, Refined Flour, Turmeric, Coriander, Chili Powder
- 💬 WhatsApp integration for orders
- 📊 Dashboard with analytics

## How to Run / कैसे चलाएं

### Method 1: Using Python
```bash
python -m http.server 8000
```
Then open: http://localhost:8000/index.html

### Method 2: Using Live Server
Open the project in VS Code and use Live Server extension
